===============================
 How to Use 1-my_list
===============================

Test the ability of making instant from class MyList
==========================
::

    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> type(my_list)
    <class '1-my_list.MyList'>

::

    >>> print(my_list)
    []

::

    >>> my_list = MyList([3])
    >>> print(my_list)
    [3]

::

    >>> my_list = MyList([3, 1, 7])
    >>> print(my_list)
    [3, 1, 7]

::

    >>> my_list = MyList(None)
    Traceback (most recent call last):
    ...
    TypeError: 'NoneType' object is not iterable

``print_sorted`` Method
=======================
::
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

::

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
